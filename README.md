## 🧱 Referência Rápida — TCP/IP vs OSI

> Esta tabela é o mapa de todos os labs: cada protocolo usado se encaixa em alguma dessas camadas.

| Camada TCP/IP | Camada OSI | Nº | Protocolos / Tecnologias |
|:---:|:---:|:---:|---|
| **Aplicação** | Aplicação | 7 | HTTP, HTTPS, FTP, DNS, DHCP, SMTP, SSH, Telnet |
| **Aplicação** | Apresentação | 6 | SSL/TLS, JPEG, ASCII, MPEG |
| **Aplicação** | Sessão | 5 | NetBIOS, RPC, PPTP |
| **Transporte** | Transporte | 4 | TCP, UDP |
| **Internet** | Rede | 3 | IP (IPv4/IPv6), ICMP, ARP |
| **Acesso à Rede** | Enlace de Dados | 2 | Ethernet, Wi-Fi 802.11, MAC |
| **Acesso à Rede** | Física | 1 | Cabo UTP, Fibra óptica, Sinal de rádio |
